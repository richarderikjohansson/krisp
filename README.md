# Kiruna Radiometric Inference and Software Platform

This is the main repository of KRISP, a software platform designed by scientists for scientists.
KRISP is primarily used for the retrieval of atmospheric trace gases from two microwave radiometers,
KIMRA and MIRA2, both operated at the Swedish Institute of Space Physics in Kiruna, Sweden.

The software is built around the PyARTS library, a comprehensive toolkit for radiative transfer calculations,
as well as simulation and retrieval applications.
